Spreadsheets

Google Sheets is an important tool for arranging numbers and doing operations with them in a convienent way that is not visually repulsive. But the table feature alone only does what HTML can do.

There's A Lot
More You Can
Do Than This.

One main feature is that you can use formulas. While HTML can do the very same thing, it's pretty easy to do. All you have to do is put in an = and then whatever operations you want. These operations involve the table cell, for example a1, and any operations you so choose. A colon will include all of the items in a sequence too. You can do stuff with this by using a function, like + or SUM().

Furthermore, conditional logic lets you do a certain thing if something happens, for example a different function depending on the data input or highlighting a different color depending on the value.
